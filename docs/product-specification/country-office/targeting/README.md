### Motivation:

The purpose of this section is to allow HCT users to draw up potential scenarios with realistic data in order to prepare Cash Plans against a target population of households and individuals. 

### How to Target:

* Name Target Set
* Set Location
* Add Rules
* Save
* Send to Cash Assist
